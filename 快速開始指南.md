## 💡 使用技巧

### 技巧1：善用CB名稱查詢
```
輸入CB名稱 → 看掛牌最高最低 → 了解歷史價格範圍
```

### 技巧2：注意理論價位
```
理論價 < 95元 → 系統可能切換到「絕對價格模式」
理論價 ≥ 95元 → 使用「溢價率模式」
```

### 技巧3：結合市場氛圍
```
強勢上升 → 系統自動 +0.7%
急劇下降 → 系統自動 -0.6%
```

### 技巧4：三種策略選擇
```
保守策略 → 接近最低得標價（75-90%得標機率）
平衡策略 → 預估價格（60-75%得標機率）
積極策略 → 確保得標（90%+得標機率）
```

## 🆘 常見問題

### Q1: 找不到CB資料？
**A:** 確保CB名稱正確，或嘗試輸入部分名稱（如「台光電」）

### Q2: 系統顯示「載入失敗」？
**A:** 檢查 `cb_data_integrated.json` 是否在同一目錄

### Q3: 想更新資料怎麼辦？
**A:** 使用 `update_cb_data.py` 腳本或找Claude重新處理

### Q4: 可以離線使用嗎？
**A:** 可以！檔案載入後完全離線運作

### Q5: 支援哪些瀏覽器？
**A:** Chrome 90+, Edge 90+, Firefox 88+, Safari 14+
